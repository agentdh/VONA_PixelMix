# VONA PixelMix

마인크래프트 스킨 제작용 데스크톱 프로그램입니다.

이 저장소는 설치 파일과 자동 업데이트 파일만 올리는 배포용 저장소입니다.
프로그램 소스 코드는 포함되어 있지 않습니다.

## 다운로드

최신 설치 파일은 아래 Releases 페이지에서 받을 수 있습니다.

https://github.com/agentdh/VONA_PixelMix/releases/latest

## 튜토리얼

프로그램 사용법은 아래 튜토리얼 페이지에서 확인할 수 있습니다.

https://agentdh.github.io/VONA_PixelMix/tutorial.html

## 설치 방법

### Windows

1. Releases 페이지에서 `VONA_PixelMix_Setup_버전.exe` 파일을 다운로드합니다.
2. 다운로드한 `.exe` 파일을 실행합니다.
3. 설치 화면 안내에 따라 설치합니다.
4. 설치가 끝나면 VONA PixelMix를 실행합니다.

### macOS

1. Releases 페이지에서 `VONA_PixelMix-버전-mac-universal.dmg` 파일을 다운로드합니다.
2. 다운로드한 `.dmg` 파일을 엽니다.
3. 앱을 Applications 폴더로 옮깁니다.
4. Applications 폴더에서 VONA PixelMix를 실행합니다.

macOS에서 "확인되지 않은 개발자" 안내가 보일 수 있습니다. 이 경우 시스템 설정의 개인정보 보호 및 보안 메뉴에서 실행을 허용하면 됩니다.

## 업데이트 방법

앱을 실행하면 새 버전이 있는지 자동으로 확인합니다.

새 업데이트가 있으면 앱 안에서 업데이트 안내가 표시됩니다.
`다운로드`를 누르면 업데이트 파일을 내려받고, 다운로드가 끝난 뒤 `재시작 후 설치`를 누르면 새 버전으로 바뀝니다.

업데이트를 바로 하지 않아도 됩니다. 나중에 앱을 다시 실행하면 다시 확인할 수 있습니다.

직접 다시 설치하고 싶다면 Releases 페이지에서 최신 설치 파일을 받아 실행하면 됩니다. 기존 설치 위에 덮어써도 됩니다.

## 릴리즈 파일 안내

Windows 사용자는 `.exe` 파일을 받으면 됩니다.

macOS 사용자는 보통 `.dmg` 파일을 받으면 됩니다. `mac-universal` 파일은 Intel Mac과 Apple Silicon Mac에서 모두 사용할 수 있습니다.

`latest.yml`, `latest-mac.yml`, `blockmap` 파일은 자동 업데이트에 필요한 파일입니다. 직접 다운로드할 필요는 없습니다.
