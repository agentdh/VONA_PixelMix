Minecraft Skin PixelMix

VONA PixelMix 사용법

슬롯에 스킨을 조립하고, 에셋 라이브러리로 관리하며, 제작실에서 직접 그리고, 명암으로 입체감을 더하는 마인크래프트 스킨 제작 툴입니다. 지금 바로 시작할 수 있도록 탭별 기능과 실전 흐름을 정리했습니다.

프로그램 개요 v1.0.2

PixelMix는 조립, 제작실, 에셋 관리, 순서 설정, 명암의 다섯 가지 탭으로 구성됩니다.

🎯 조립

여러 슬롯(머리카락·상의·하의 등)에 스킨 에셋을 배치해 최종 스킨을 미리보고 PNG 또는 .skinproj로 내보냅니다.

📦 에셋 관리

슬롯별 스킨 파일을 라이브러리로 관리합니다. 에셋을 등록하면 조립 탭과 제작실에서 바로 불러올 수 있습니다.

🧩 제작실

에셋을 선택해 픽셀 단위로 직접 그립니다. 레이어·폴더·다양한 도구를 지원하며 .skinproj로 저장합니다.

📌
기본 흐름 요약에셋 관리에서 슬롯마다 스킨을 등록 → 조립 탭에서 최종 스킨 미리보기 → 제작실에서 픽셀 편집 → PNG / .skinproj 내보내기. 스킨 모델은 Wide와 Slim으로 구분되므로 등록 시 맞는 모델을 선택하세요.

🧩 조립 조립 탭

조립 탭에서는 각 슬롯에 에셋을 배치해 최종 스킨을 완성합니다. 3D 미리보기로 실시간 확인이 가능합니다.

📂 슬롯 구성

악세서리·세트의상·상의·하의·신발·머리카락·얼굴·피부 순으로 슬롯이 배치됩니다. 슬롯 설정 탭에서 순서와 공유 여부를 변경할 수 있습니다.

🎯 스킨 선택

각 슬롯에서 등록된 에셋을 선택합니다. Wide·Slim 모델별로 별도 선택이 가능하며 동시에 표시됩니다.

🖼️ 미리보기

선택된 에셋들이 2D 전개도와 3D 프리뷰에서 합성되어 표시됩니다. Wide·Slim 공유 슬롯은 하나의 이미지가 두 모델에 모두 적용됩니다.

🌗 명암 적용

명암 탭에서 등록한 명암 설정이 조립 스킨에 실시간으로 반영됩니다.

조립 탭 저장 옵션

조립 탭에서 .skinproj 내보내기와 PNG 저장 버튼이 나란히 배치되어 있습니다. .skinproj는 슬롯 구성 정보를 포함하며, PNG는 현재 전개도에 표시된 합성 결과를 저장합니다.

⚠️
조립 탭 저장 시 주의조립 탭의 .skinproj를 제작실에서 불러오면 슬롯 구조만 레이어로 가져오며, 명암 설정은 포함되지 않습니다.
🎨 제작실 제작실 탭

제작실은 픽셀 단위로 스킨을 직접 그리는 편집기입니다. 에셋을 선택해야 편집이 활성화됩니다.

🔒
에셋 선택 후 편집 가능프로그램을 처음 실행하거나 재시작하면 제작실은 잠금 상태입니다. 좌측 패널에서 에셋을 선택해야 레이어 편집과 모든 도구가 활성화됩니다.
📐 레이아웃

기본 레이아웃은 좌측 전개도 · 우측 3D 프리뷰입니다. 상단 프리뷰 조정 버튼으로 배치를 바꿀 수 있습니다. 도구 선택 패널은 프리뷰 조정 바 아래에 위치합니다.

🖊️ 도구 선택

연필·지우개·직선·영역·도장·연결채우기 여섯 가지 도구를 지원합니다. 각 도구에 단축키가 지정되어 있습니다.

📑 레이어

레이어 추가, 폴더 추가, 드래그로 순서 변경, 우클릭으로 복제·삭제·아래 레이어와 합치기가 가능합니다. 폴더를 드래그해서 레이어를 폴더 안으로 이동할 수 있습니다.

🗑️ 레이어 지우기

편집 버튼(✨)과 ADV 버튼 사이의 🗑️ 버튼을 누르면 레이어 내용만 초기화됩니다. 레이어 자체는 유지됩니다.

🖼️ 마네킹 제어

우측 패널에서 마네킹 활성/비활성, 이미지 업로드, 불투명도를 설정할 수 있습니다. 마네킹은 참조용으로만 사용되며 PNG 저장에는 포함되지 않습니다.

🧱 파츠 표시

우측 패널 상단의 파츠 표시 패널에서 모자·갑옷·팔갑옷 등 각 파트를 개별적으로 켜고 끌 수 있습니다.

우측 패널 구성 (위→아래)

패널기능
파츠 표시3D·전개도에서 표시할 신체 파트 개별 토글
마네킹 제어참조용 마네킹 이미지 설정 및 불투명도 조절
레이어 목록레이어·폴더 추가, 드래그 정렬, 편집/지우기/ADV, 우클릭 메뉴
패턴 제작현재 레이어에서 패턴 생성 및 적용
컬러색상 피커 (RGBA / HSVA)
💡
레이어 편집기(✨)편집 버튼을 누르면 정밀 편집기가 열립니다. 영역 선택·채우기·색 선택 도구와 Hue·Brightness·Contrast 보정, 레이어 효과(일반·곱하기·오버레이·발광)를 적용할 수 있습니다. 폴더에서 편집을 열면 영역·채우기·색 선택 도구는 표시되지 않습니다.
📦 에셋 관리 에셋 관리 창

에셋 관리 창에서 슬롯별로 스킨 파일을 등록하고 관리합니다.

📁 에셋 등록

폴더별로 에셋을 분류하고 이름·아이콘을 지정합니다. 등록된 에셋은 조립 탭에서 즉시 선택 가능합니다.

⭐ 즐겨찾기

자주 쓰는 에셋은 즐겨찾기로 등록해 빠르게 접근할 수 있습니다.

📤 불러오기

PNG, JSON, .skinproj 파일을 가져올 수 있습니다. .skinproj는 레이어 구조까지 복원되며, 조립 탭 .skinproj는 슬롯 선택 정보를 불러옵니다.

🖥️ 편집 연동

에셋 편집 버튼을 누르면 해당 에셋이 제작실에 로드되어 바로 편집할 수 있습니다.

↕️ 순서 순서 설정 탭

순서 설정 탭에서는 슬롯의 합성 순서를 조절합니다. 위쪽 슬롯일수록 최종 스킨에서 위에 그려집니다.

항목설명
기본 슬롯 순서악세서리→세트의상→상의→하의→신발→머리카락→얼굴→피부 (위에서 아래로)
공유 슬롯하의·신발·머리카락·얼굴은 기본적으로 Wide/Slim 공유가 활성화됩니다.
폴더/에셋 순서슬롯 안에서 에셋 우선순위를 드래그로 조정할 수 있습니다.
🌗 명암 명암 탭

명암 탭은 조립된 스킨에 명암 파일을 합성해 입체감을 더합니다.

🖼️ 명암 파일 등록

Wide·Slim 모델별로 명암 가이드 이미지를 등록합니다. 밝은 영역일수록 강하게 적용됩니다.

📂 명암 프리셋

명암 파일을 활성화하면 조립 스킨(A) 위에 명암(B)이 클리핑되어 합성됩니다.

🎨 색조 보정

Hue Shift·밝기·Luminance를 조절해 명암의 색감을 변경할 수 있습니다.

✨ 블렌드 모드

곱하기·오버레이·발광을 활성화해 명암의 합성 방식을 선택합니다. 여러 개 동시 적용 가능합니다.

🧩
명암 합성 구조명암 파일 B를 마스크로 스킨 A를 클리핑 → Hue·Brightness·Luminance 보정 적용 → 곱하기·오버레이·발광 블렌드로 A 위에 합성. 명암 파일이 여러 개일 경우 각각 독립적으로 처리 후 누적 합성됩니다.
저장과 불러오기

PixelMix는 PNG와 .skinproj 두 가지 형식을 사용합니다.

형식저장 위치내용
PNG조립 / 제작실현재 전개도에 표시된 레이어 합산 결과. 마네킹은 포함되지 않습니다.
제작실 .skinproj제작실레이어 구조·내용·설정 전체를 보존합니다.
조립 .skinproj조립 탭슬롯 선택 구성, 명암 설정, 순서를 포함합니다.
📌
버튼 위치조립 탭의 .skinproj 내보내기와 PNG 저장 버튼은 나란히 반반 크기로 배치되어 있습니다. 제작실의 PNG 저장은 좌측 패널 상단에 위치합니다.

따라하기 체크리스트

아래 순서대로 따라하면 처음 사용자도 빠르게 PixelMix에 익숙해질 수 있습니다.

  1. 제작실 탭을 열고 에셋을 선택합니다. 에셋 관리에서 편집할 에셋을 선택하면 잠금이 해제되어 레이어 편집이 활성화됩니다.

  2. 제작실에서 스킨을 그립니다. 레이어를 추가하고 도구를 선택해 픽셀을 그립니다. 편집이 끝나면 .skinproj로 내보내거나 에셋 저장으로 반영합니다.

  3. 레이어 기능을 활용합니다. 레이어 우클릭으로 복제·삭제·합치기, 편집(✨)·🗑️·ADV 버튼을 사용해보세요.

  4. 조립 탭에서 슬롯에 에셋을 배치합니다. 3D 프리뷰로 전체 스킨을 확인하고 .skinproj 또는 PNG로 내보냅니다.

  5. 명암 탭을 활용합니다. 명암 파일을 등록하고 Hue·밝기·블렌드 모드를 조절해 입체감 있는 스킨을 완성합니다.

  6. 최종 파일을 저장합니다. 완성 후 PNG 저장으로 스킨을 내보내고, 조립 탭 .skinproj로 전체 구성을 백업합니다.

제작실에서 에셋을 선택해 레이어 편집이 활성화되는지 확인했다.
레이어 추가·우클릭 메뉴·🗑️ 지우기 버튼을 사용해봤다.
조립 탭에서 슬롯에 에셋을 배치하고 .skinproj와 PNG 내보내기를 해봤다.
명암 탭에서 명암 파일을 등록하고 블렌드 모드를 적용해봤다.
