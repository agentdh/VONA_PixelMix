VONA PixelMix 튜토리얼

3D 프리뷰와 전개도에서 스킨 에셋을 만들고, 부위별 에셋을 조립해 여러 스킨을 제작하는 흐름을 중심으로 설명합니다.

프로그램 개요 CURRENT

PixelMix는 제작실에서 레이어 기반 에셋을 만들고, 조립 탭에서 슬롯별 에셋을 선택해 완성 스킨을 만드는 편집 도구입니다.

✏️ 제작

전개도와 3D 프리뷰를 보며 직접 그리고, 레이어·패턴·효과를 사용해 에셋을 만듭니다.

🧩 조립

악세서리, 의상, 신발, 머리카락, 얼굴, 피부 등 슬롯별 에셋을 골라 하나의 스킨으로 합칩니다.

🎨 그룹 편집

같은 색상 그룹명을 공유하는 레이어만 묶어 색조 조절과 팔레트 치환을 적용합니다.

◐ 자동 명암

조립된 스킨의 색과 명암 마스크의 알파를 결합해 색조·채도·명도·휘도·불투명도를 조절합니다.

권장 작업 흐름

처음에는 아래 순서를 따라가면 제작과 조립의 관계를 빠르게 익힐 수 있습니다.

에셋 선택 또는 생성→제작실 편집→에셋 저장→슬롯 설정 확인→조립→명암 적용→내보내기

제작실

제작실은 하나의 에셋을 레이어 단위로 편집하는 공간입니다. 먼저 에셋 관리에서 작업할 에셋을 선택해야 편집 도구가 활성화됩니다.

📁 에셋 관리

새 프리셋을 만들거나 기존 에셋을 선택해 제작실로 불러옵니다. 선택된 에셋이 없으면 저장 버튼과 편집 기능은 조용히 비활성화됩니다.

🗺 전개도 편집

연필, 지우개, 직선, 영역 채우기, 연결 색상 채우기, 도장 도구를 사용합니다. 휠 또는 지정된 이동 조작으로 화면을 이동하고 확대할 수 있습니다.

🧍 3D 직접 편집

3D 그리기 모드를 켜면 선택한 레이어의 보이는 부위를 직접 그릴 수 있습니다. 우클릭은 모델 회전에 사용되며 도구 미리보기도 3D 표면에 표시됩니다.

🧰 패널 배치

패널은 헤더를 누르고 드래그할 때만 이동합니다. 색상환과 패턴 제작 패널은 하나의 영역으로 묶여 높이를 조절할 수 있습니다.

💡
에셋을 먼저 선택하세요프로그램을 다시 실행하면 제작실은 에셋 미선택 상태로 시작합니다. 에셋을 선택하기 전에는 전개도, 3D 그리기, 레이어 추가·삭제와 효과 조작이 적용되지 않습니다.

레이어와 패턴

▤ 레이어 목록

레이어 추가·삭제, 표시, 순서 변경, 폴더 정리, 편집과 ADV 설정을 관리합니다. 지우기 버튼은 해당 레이어의 내용을 비웁니다.

◫ 클리핑

클리핑된 레이어는 바로 아래 레이어의 불투명 영역 안에서만 보입니다.

✨ 레이어 효과

일반, 곱하기, 오버레이, 발광 효과를 선택하고 색조·채도·명도를 조절할 수 있습니다.

🧱 패턴 제작

선택 레이어에서 패턴을 만들고 X/Y 복제 수, 간격, 이동, 개별 회전과 전체 회전을 조절합니다.

🎨
패턴 팔레트 치환패턴 색상 칸을 더블클릭하면 색상 치환 패널이 열립니다. 색상을 변경하거나 패널의 변경 초기화 버튼으로 해당 색상만 원래대로 돌릴 수 있습니다.

조립 탭

제작실에서 저장한 에셋을 슬롯별로 선택해 하나의 스킨을 만드는 화면입니다.

🧩 슬롯 선택

각 슬롯에서 사용할 에셋을 선택하면 전개도와 3D 프리뷰에 즉시 반영됩니다. 다중 선택을 지원하는 슬롯은 여러 에셋을 함께 사용할 수 있습니다.

↕ 합성 순서

슬롯 설정의 순서와 에셋의 겹침 설정에 따라 최종 합성 순서가 결정됩니다.

🎨 색상 그룹

현재 조립에 포함된 색상 그룹을 편집합니다. 결과는 해당 그룹 레이어가 포함된 슬롯에만 나타납니다.

💾 스킨 저장

PNG는 완성 이미지로, 조립용 .skinproj는 슬롯 선택·순서·공유·명암 설정을 포함한 프로젝트로 저장합니다.

색상 그룹 편집

같은 색상 그룹명을 가진 레이어들만 하나의 편집 대상으로 묶습니다.

  1. 그룹 선택
    조립 탭의 색상 그룹 목록에서 편집할 그룹명을 선택합니다.
  2. 중복 색상 통합
    그룹에 포함된 모든 선택 에셋의 레이어 색상을 수집하며, 같은 색상은 팔레트에 한 번만 표시합니다.
  3. 더블클릭 치환
    팔레트 색상을 더블클릭해 색상 치환 패널을 열고 새 색상을 지정합니다. 변경 초기화로 해당 색상만 복원할 수 있습니다.
  4. 그룹 범위 적용
    같은 원본 색상이 다른 그룹이나 그룹이 없는 레이어에 있어도 변경되지 않습니다. 현재 그룹을 사용하는 슬롯의 결과만 바뀝니다.

명암 시스템

명암 탭에서 등록한 마스크를 조립 탭의 명암 레이어 설정에서 활성화합니다.

◐ 명암 마스크

명암 파일 B의 알파 모양에 조립 스킨 A의 색을 넣어 C 명암 레이어를 만듭니다. C는 A 위의 최상단에 적용됩니다.

☀ 속성 조절

색조, 채도, 명도, 휘도, 불투명도는 B나 A 하나가 아니라 완성된 C 명암 레이어 전체에 적용됩니다.

✦ 블렌딩

곱하기, 오버레이, 발광을 켜면 C 레이어에 해당 효과가 적용됩니다.

▣ 일반 레이어 유지

세 블렌딩 효과를 모두 꺼도 명암이 사라지지 않습니다. 조절값이 적용된 C가 일반 레이어로 최상단에 남습니다.

슬롯 설정

기본 합성 순서는 악세서리 → 세트의상 → 상의 → 하의 → 신발 → 머리카락 → 얼굴 → 피부입니다.

설정역할
순서앞쪽 슬롯이 최종 합성에서 더 위에 보이도록 순서를 정리합니다.
공유Wide와 Slim 모델에서 같은 선택을 공유합니다. 기본 공유 슬롯은 하의, 신발, 머리카락, 얼굴입니다.
겹침 설정특정 레이어를 최상단으로 올리는 등 에셋 간 비정상적인 겹침을 방지합니다.

저장과 불러오기

형식용도
PNG완성된 스킨 이미지 또는 한 장의 레이어로 가져올 이미지입니다.
제작실 .skinproj레이어, 폴더, 패턴, 색상과 효과 등 제작실 편집 구조를 저장합니다.
조립 .skinproj슬롯 조합, 순서, 공유 설정과 명암 설정을 저장합니다. 제작실로 가져오면 슬롯 합성 구조만 레이어로 변환되며 명암 구조는 제외됩니다.
기존 JSON이전 버전 파일을 불러오기 위한 호환 형식입니다. 새 작업은 .skinproj 사용을 권장합니다.
📂
없는 슬롯 자동 복원조립 프로젝트를 불러올 때 저장 당시 사용한 슬롯이 현재 없다면, 필요한 슬롯을 목록의 가장 아래에 자동으로 추가합니다.

정규버전 코드 입력

처음 설치하면 PIXELMIX(beta) 상태로 시작합니다. 설정 탭의 정규버전 코드 입력란에 유효한 코드를 입력하면 잠금이 해제되고, 재실행 후에도 상태가 유지됩니다. 설정에서 다시 잠글 수 있습니다.

항목코드 입력 후 해제되는 제한
슬롯 관리폴더와 슬롯 생성 기능을 사용할 수 있고 파일 시스템 변경도 프로그램에 반영됩니다.
패턴 편집X/Y 복제 수 제한, 복제 간격 제한과 개별 타일 회전 잠금이 해제됩니다.
명암 탭프로그램 안에서 명암 파일을 추가·교체·제거할 수 있습니다.
조립 명암명도와 휘도를 100% 이외의 값으로 조절할 수 있습니다.
제작실 레이어5개를 넘는 레이어를 사용할 수 있습니다.
조립 프로젝트조립 결과를 단일 레이어가 아닌 슬롯별 구조로 보존할 수 있습니다.
🔐
코드는 별도로 제공됩니다이 가이드에는 정규버전 코드 또는 비밀번호를 기재하지 않습니다. 제공받은 유효한 코드만 설정 화면에 입력하세요.

따라하기

  1. 에셋 관리에서 새 프리셋을 만들거나 기존 에셋을 선택합니다.
  2. 제작실에서 레이어를 추가하고 전개도 또는 3D 그리기로 색을 넣습니다.
  3. 패턴 레이어를 만들고 팔레트 색상을 더블클릭해 색을 치환해 봅니다.
  4. 에셋을 저장한 뒤 조립 탭에서 해당 슬롯의 에셋으로 선택합니다.
  5. 같은 색상 그룹을 가진 레이어가 있다면 그룹 팔레트에서 색을 바꾸고 다른 그룹은 변하지 않는지 확인합니다.
  6. 명암 마스크를 활성화하고 블렌딩 효과를 모두 끈 뒤에도 일반 명암 레이어가 남는지 확인합니다.
  7. 완성 스킨은 PNG로, 다시 편집할 조립 상태는 .skinproj로 내보냅니다.
