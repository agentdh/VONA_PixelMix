Minecraft Skin PixelMix

VONA PixelMix 튜토리얼

처음 쓰는 사람도 슬롯, 에셋, 제작실, 조립, 명암의 역할을 빠르게 이해할 수 있도록 최신 버전 기준으로 정리했습니다. 설명은 내부 코드가 아니라 화면 구성과 작업 논리 중심으로 안내합니다.

프로그램 개요 v1.0.3_13 Snapshot

PixelMix는 마인크래프트 스킨을 한 장의 그림으로만 다루지 않고, 슬롯과 에셋, 레이어, 명암 설정으로 나누어 조립하는 제작 도구입니다.

🎯 슬롯

악세서리, 상의, 피부처럼 스킨을 구성하는 위치입니다. 슬롯 순서가 최종 조립 순서에 영향을 줍니다.

📦 에셋

슬롯 안에 저장되는 후보 이미지입니다. 제작실에서 레이어 구조로 만들고 에셋 관리에 저장합니다.

🧩 조립 스킨

여러 슬롯에서 선택한 에셋을 합친 결과입니다. PNG로 저장하거나 .skinproj로 보관할 수 있습니다.

📌
기본 슬롯 순서최신 기준 기본 순서는 악세서리 → 세트의상 → 상의 → 하의 → 신발 → 머리카락 → 얼굴 → 피부입니다. 하의, 신발, 머리카락, 얼굴은 Wide와 Slim에서 공유 슬롯으로 시작합니다.
기본 작업 흐름

처음에는 제작실에서 에셋을 만들고, 에셋 관리에 저장한 뒤, 조립 탭에서 슬롯별로 선택해 최종 스킨을 만드는 흐름을 추천합니다.

🧩 조립 조립 탭

조립 탭은 슬롯별 에셋을 선택하고, 3D 미리보기와 최종 저장을 확인하는 중심 화면입니다.

📂 프로젝트 패널

새 조립 스킨 이름을 정하고, 현재 조합을 PNG 또는 .skinproj로 내보냅니다. 저장된 스킨 목록도 이 영역에서 관리합니다.

🎯 슬롯 선택

현재 선택된 슬롯에 들어갈 에셋을 고릅니다. 공유 슬롯은 Wide와 Slim 선택이 함께 유지됩니다.

🖼️ 미리보기

선택한 에셋들이 순서대로 합쳐진 결과를 2D와 3D로 확인합니다. Wide와 Slim 모델을 바꾸며 확인할 수 있습니다.

🌗 명암 레이어 설정

활성화한 명암 파일을 최종 조립 스킨 위에 겹쳐 보정합니다. 명암 색상은 조립 스킨 색을 바탕으로 만들어집니다.

조립 탭 저장 방식

조립 탭에서 저장한 .skinproj는 조립 탭 전용 정보를 함께 담습니다. 다시 조립 탭에서 불러오면 슬롯 선택 상태 그대로 복원되고, 제작실에서 불러오면 슬롯 합성 결과가 편집 가능한 레이어 구조로 변환됩니다. 이때 명암 관련 구조는 제작실 레이어에서는 제외됩니다.

⚠️
조립 탭 전용 파일 확인조립 탭에서 저장한 .skinproj가 아닌 파일을 조립 탭에 불러오면 경고가 표시되고 불러오기가 중단됩니다.
🎨 제작실 제작실 탭

제작실은 에셋을 직접 편집하는 공간입니다. 에셋관리에서 에셋을 선택해야 레이어와 도구 조작이 열리며, 선택 전에는 저장 버튼이 비활성화되고 편집 도구도 잠깁니다.

📦 에셋관리 / 저장

에셋관리 버튼으로 라이브러리 창을 열고 수정할 에셋을 선택합니다. 저장 버튼은 선택된 에셋을 덮어씌웁니다.

🗺️ 전개도 패널

스킨 전개도를 픽셀 단위로 편집합니다. 휠 또는 우클릭 드래그로 화면을 이동하고, 손을 떼면 이동 상태가 끝납니다.

👤 3D 프리뷰

현재 레이어 조합이 실제 캐릭터 모델에서 어떻게 보이는지 확인합니다. 모델과 보이는 부위를 바꿀 수 있습니다.

🧱 레이어 목록

레이어 추가, 폴더 추가, 표시/숨김, 클리핑, ADV 보정, 편집, 지우기를 관리합니다. 편집과 ADV 사이에는 🗑️ 지우기 버튼이 있습니다.

🧩 패턴 제작

선택 레이어에서 반복 패턴을 만들고 적용합니다. 반복 개수, 간격, 회전 등으로 타일 배치를 조정합니다.

🎚️ 레이아웃 조정

패널의 최상단 헤더를 잡고 드래그하면 패널을 이동할 수 있습니다. 바 조작과 색상환 조작은 해당 조작만 수행되도록 분리되어 있습니다.

💡
재실행 후 제작실 상태프로그램을 다시 열면 제작실은 에셋 미선택 상태로 시작합니다. 에셋을 고르기 전에는 연필, 영역 채우기, 레이어 추가/삭제, 효과 변경 같은 조작을 할 수 없습니다.
📦 에셋 관리 에셋 관리 창

에셋 관리는 독립 탭이 아니라 제작실에서 여는 관리 창입니다. 슬롯, 폴더, 프리셋을 관리하고 선택한 에셋을 제작실로 불러옵니다.

📁 슬롯 관리

폴더와 슬롯을 만들고 이름, 아이콘, 위치를 조정합니다. 잠금 상태에서는 폴더와 슬롯 생성이 제한됩니다.

➕ 새 프리셋 추가

빈 레이어 1개를 가진 새 프리셋을 만듭니다. 이름과 대상 모델을 정한 뒤 저장하면 해당 슬롯에 추가됩니다.

📥 불러오기

PNG, JSON, .skinproj를 가져올 수 있습니다. PNG는 단일 레이어 에셋으로, 제작실 파일은 레이어 구조 그대로, 조립 파일은 슬롯 합성 구조를 레이어로 변환해 가져옵니다.

🖱️ 선택과 편집

에셋을 클릭하면 미리보기와 편집 패널이 열립니다. 제작실로 불러오기를 누르면 해당 에셋이 제작실에서 수정 가능한 상태가 됩니다.

↕️ 순서 순서 설정 탭

순서 설정 탭은 슬롯이 쌓이는 순서와 공유 여부를 정하는 곳입니다. 위아래 순서가 달라지면 최종 합성 결과도 달라집니다.

항목역할
렌더 순서최종 스킨을 합칠 때 어떤 슬롯이 위에 올라오는지 결정합니다.
공유 선택Wide와 Slim 모델에서 같은 슬롯 선택을 공유할지 정합니다.
폴더/슬롯 관리슬롯을 폴더로 묶거나 이름, 아이콘, 위치를 정리합니다.
🌗 명암 명암 탭

명암 탭은 명암 파일과 가이드 이미지를 관리하고, 조립 탭에서 사용할 명암 소스를 준비하는 공간입니다. 잠금 상태에서는 명암 탭 전체가 잠깁니다.

🖼️ 가이드 등록

Wide와 Slim 모델별로 명암 마스크를 편집할 때 참고할 가이드 이미지를 등록합니다.

📂 명암 마스크

명암 파일을 추가하고 이름을 바꾸거나 삭제합니다. 실제 적용은 조립 탭의 명암 레이어 설정에서 켜고 조절합니다.

🎚️ 조절 값

불투명도, 색조, 채도, 명도, 휘도를 조절합니다. 값은 명암 파일 단독이 아니라 조립 스킨 색이 들어간 명암 묶음 전체에 적용됩니다.

✨ 효과

곱하기, 오버레이, 발광을 켜면 명암 묶음에 레이어 효과처럼 적용됩니다. 여러 효과를 함께 켤 수 있습니다.

🧠
명암 적용 이해A 스킨과 B 명암이 있을 때, B 명암의 알파 모양 안에 A 스킨의 색을 넣어 C 묶음을 만들고, 그 C 묶음을 다시 A 스킨 위에 클리핑해 올리는 방식으로 생각하면 됩니다. 색조, 채도, 명도, 휘도, 불투명도와 효과는 이 C 묶음에 적용됩니다.
잠금 상태와 정규 상태

처음 설치하면 프로그램 제목은 PIXELMIX(beta)로 표시됩니다. 설정 탭의 잠금 메뉴에서 비밀번호를 입력하면 PIXELMIX 상태로 바뀌고, 재실행 후에도 유지됩니다.

잠금 대상잠금 상태 동작
제작실 슬롯/폴더 생성프로그램 안에서 생성할 수 없고, 파일 탐색기 변경도 리로드되지 않습니다.
패턴 편집X/Y 복제 개수는 10으로 고정되고, 복제 간격은 -5에서 +5 사이로 제한되며, 개별 타일 회전은 잠깁니다.
명암 탭탭 전체가 잠기며 프로그램 안에서 명암 파일 추가, 교체, 제거를 할 수 없습니다.
조립 명암명도와 휘도는 100%로 고정됩니다.
제작실 레이어최대 5개까지만 사용할 수 있습니다.
조립 저장잠금 상태에서는 슬롯별 구조가 아니라 하나의 합성 레이어처럼 저장됩니다.
저장과 불러오기

PixelMix는 목적에 따라 PNG와 .skinproj를 나누어 사용합니다. .skinproj 표기는 최신 기준으로 소문자를 사용합니다.

파일사용 위치내용
PNG조립 / 제작실 / 에셋 관리최종 이미지 또는 단일 레이어 이미지입니다.
제작실 .skinproj제작실레이어, 패턴, 보정, 제작실 설정을 포함한 편집용 프로젝트입니다.
조립 .skinproj조립 탭슬롯 선택 구조, 순서, 공유, 명암 설정을 포함한 조립용 프로젝트입니다.
JSON호환용기존 파일을 불러오기 위한 호환 형식입니다. 최신 작업은 .skinproj를 권장합니다.

따라하기 체크리스트

아래 순서대로 한 번만 따라 하면 PixelMix의 전체 흐름을 거의 다 써볼 수 있습니다.

  1. 설정에서 잠금 상태를 확인합니다. 필요한 경우 잠금 메뉴에서 비밀번호를 입력해 정규 상태로 전환합니다.

  2. 제작실로 이동합니다. 에셋관리 버튼을 눌러 수정할 에셋을 선택합니다. 선택 전에는 저장 버튼과 편집 조작이 잠긴 상태입니다.

  3. 레이어를 편집합니다. 연필, 영역 채우기, 도장, 패턴 제작을 써보고 레이어 목록의 편집, 지우기, ADV 버튼을 확인합니다.

  4. 저장 버튼을 누릅니다. 선택한 에셋을 덮어씌웁니다. 별도 프로젝트로 보관하고 싶으면 .skinproj 내보내기를 사용합니다.

  5. 조립 탭에서 슬롯별 에셋을 고릅니다. 3D 프리뷰를 보면서 필요한 슬롯을 채웁니다.

  6. 명암을 적용합니다. 명암 파일을 활성화하고 색조, 채도, 명도, 휘도, 불투명도와 효과를 조절합니다.

  7. 최종 파일을 저장합니다. 바로 사용할 스킨은 PNG로, 다시 편집할 조립 상태는 .skinproj로 내보냅니다.

에셋 선택 전 제작실 조작이 잠기는지 확인했다.
새 프리셋 추가 후 빈 레이어 1개가 생기는지 확인했다.
제작실 저장은 선택 에셋을 덮어씌우고, .skinproj 내보내기는 별도 파일을 만드는 차이를 이해했다.
조립 .skinproj와 제작실 .skinproj의 용도가 다르다는 점을 확인했다.
